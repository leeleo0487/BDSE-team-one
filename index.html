Repeat Buyers Prediction-Challenge the Baseline

Motivation

Problem Definition

from organizer

It is important for merchants to identify who can be converted into repeated buyers. By targeting on these potential loyal customers, merchants can greatly reduce the promotion cost and enhance the return on investment (ROI). It is well known that in the field of online advertising, customer targeting is extremely challenging, especially for fresh buyers. However, with the long-term user behavior log accumulated by Tmall.com, we may be able to solve this problem.

A set of merchants and their corresponding new buyers acquired during the promotion on the "Double 11" day. Our task is to predict which new buyers for given merchants will become loyal customers in the future. Here is the data field:
  • User id
  • Age range
  • Gender
  • Merchant id
  • Activity log

Descriptions

Data Sources

The dataset comes from the Alibaba Cloud Tianchi competitions, and it is related to user behavior log accumulated by Tmall.com. The analysis goal is to predict the probability that these new buyers would purchase items from the same merchants again within 6 months.

The dataset can be downloaded from here.

ETL

Cleaning and manipulating data with Python.

Data Warehouse

We are going to build two system to store our data. One is hadoop ecosystem, the other is relational database. That we can compare our two systems' performances.

Analyses

In pilot study, we have tried some data exploration and descriptive statistic. To understand how these data profile and help us to understand of key characteristics, that we can use to create derivative column to build our model.

Visualization

We chouse Django to be our Web framework to make our development more clean and pythonic. By it's work on server, to connect our database to web. On the other hand, we use Plotly and Ajax to make our result more impressive.
